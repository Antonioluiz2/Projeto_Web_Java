# Projeto_Web_Java
Cadastro de Vaga de emprego com Java &amp; Spring Boot

Aplifcação simples para controle de cadastro web de vagas de emprego com armaenamento em BD.

Tecnologias utilizadas:
Java 11
Spring Boot
Spring Web
Spring Data JPA
Maven
Lombok
Banco de Dados MySQL
